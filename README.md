# Countries Web Application

Welcome to the Countries Web Application! This application allows you to explore information about different countries, including population, languages, currencies, and more.

## Site

The application is available at: [https://valb268.github.io/countries/](https://valb268.github.io/countries/)

## Technologies Used

- React.js: A JavaScript library for building user interfaces.
- Bootstrap: A popular front-end framework for responsive and mobile-first websites.
- Chart.js: A JavaScript library for creating charts and visualizations.

## Installation

To run the application locally, follow these steps:

1. Clone the repository to your local machine using the following command:

git clone https://github.com/valb268/countries.git


2. Navigate to the project directory:

cd countries


3. Install the dependencies by running the following command:

npm install


## Running the Application

After completing the installation steps, you can run the application using the following command:

npm start


This command starts the development server and opens the application in your default web browser. If the browser doesn't open automatically, you can access the application by visiting [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

- Upon opening the application, you will see a dropdown button labeled "Show charts".
- Click on the dropdown button to reveal a list of available charts.
- Each chart option represents different data visualizations related to countries.
- Click on a chart option to display a modal window with the selected chart.
- Close the modal window by clicking outside the chart or using the close button.
